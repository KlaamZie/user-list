# Abc employee list

## Description

This is a list of employees of Abc company.

## Installation

1. Clone the repo
2. Run `npm install`
3. Run `npm start`
4. Open `http://localhost:1234/` in your browser

## Features

1. List of employees
2. Pagination
3. Filter by nationality

## Technologies

1. Parcel
2. React

## API

1. https://randomuser.me/
